# 🛒 Sistema de Caixa de Supermercado

Este projeto foi desenvolvido como parte da disciplina de Engenharia de Software, com o objetivo de aplicar na prática conceitos de desenvolvimento de sistemas e organização de projetos.

A aplicação simula o funcionamento de um caixa de supermercado, permitindo registrar produtos, visualizar os itens adicionados e acompanhar o valor total da compra. O sistema busca representar, de forma simples, o fluxo básico de atendimento em um caixa, desde a inserção dos produtos até a confirmação do pagamento.

Durante o desenvolvimento do projeto, foram aplicados conceitos importantes da engenharia de software, como estruturação do sistema, modularização de componentes, organização do código e uso de ferramentas modernas de desenvolvimento.

## 🚀 Tecnologias utilizadas

* React

* JavaScript

* Vite

* HTML

* CSS

### 🎯 Objetivo do projeto

O objetivo deste projeto é demonstrar, de forma prática, a construção de uma aplicação web simples utilizando tecnologias modernas do desenvolvimento front-end, além de reforçar conceitos estudados na disciplina de Engenharia de Software.


# 📦 Guia de Comandos Git do Projeto

Este documento mostra os principais comandos do **Git** usados no desenvolvimento do projeto.

---

## 📥 Clonar o repositório (primeira vez)

Usado quando você ainda **não tem o projeto no computador**.

```
git clone URL_DO_REPOSITORIO
```

Depois entre na pasta:

```
cd nome-do-projeto
```

---

## 🔄 Atualizar o projeto

Baixa a **versão mais recente do repositório**.

```
git pull 
```

Use esse comando **antes de começar a programar**, para evitar conflitos.

---

## 📁 Verificar arquivos modificados

Mostra quais arquivos foram alterados.

```
git status
```

---

## ➕ Adicionar arquivos para commit

Adicionar todos os arquivos modificados:

```
git add .
```

Adicionar apenas um arquivo específico:

```
git add nome-do-arquivo
```

---

## 💾 Salvar alterações (commit)

Salva suas mudanças localmente.

```
git commit -m "Descrição da alteração"
```

Exemplo:

```
git commit -m "Adiciona componente de pagamento"
```

---

## 🚀 Enviar alterações para o GitHub

Envia seus commits para o repositório online.

```
git push origin main
```

---

## 🔁 Fluxo de trabalho recomendado

Sempre siga esta ordem:

```
git pull
git add .
git commit -m "descrição da mudança"
git push
```

---


### 🛠 Comando útil

Ver histórico de commits:

```
git log
```
